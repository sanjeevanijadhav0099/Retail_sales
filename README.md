# Retail_sales
This project analyzes an online retail sales dataset to uncover key business insights through exploratory data analysis (EDA). Using SQL for data querying and Python (Pandas, Matplotlib, Seaborn) for processing and visualization, I explore customer behavior, sales trends, product performance, and revenue patterns. Ideal for demonstrating real-world data analytics skills in retail and e-commerce.Dataset: Online Retail dataset  covering transactions from a UK-based online retailer (2010-2011), with ~500k records including invoice details, stock codes, quantities, prices, and customer IDs.
